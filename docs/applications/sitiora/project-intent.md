# Sitiora - Intencionalidad del Proyecto

Last updated: 2026-03-04

## 1) Que es Sitiora
Sitiora es la oferta comercial de Kumera para venta de desarrollo web orientado a conversion de negocio.

## 2) Proposito
Captar empresas y profesionales que necesitan:
- Landing pages
- Sitios web corporativos
- Proyectos web de mayor complejidad bajo cotizacion

## 3) Modelo comercial inicial
- Estrategia lead-first: formulario + WhatsApp.
- Cierre comercial manual.
- Formalizacion contractual y pago via stack de billing en Kumera Platform.

## 4) Oferta y pricing
Niveles publicos iniciales:
1. Start (entrada): desde $79.000 CLP + IVA.
2. Growth (volumen): desde $149.000 CLP + IVA.
3. Pro (ticket medio): desde $249.000 CLP + IVA.
4. Custom: desde $490.000 CLP + IVA.
5. E-commerce: desde $1.290.000 CLP + IVA.

## 5) Principios operativos
1. Claridad de alcance antes de presupuesto final.
2. Planes estandar para cierre rapido.
3. Plan custom para proyectos fuera de catalogo.
4. Trazabilidad en billing por servicio `sitiora`.

## 6) KPI iniciales
- Leads recibidos por formulario.
- Leads recibidos por WhatsApp.
- Tasa lead -> propuesta.
- Tasa propuesta -> cierre.
- Tiempo promedio de cierre.
