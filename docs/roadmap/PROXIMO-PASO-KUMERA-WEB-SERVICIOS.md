# Proximo Paso: Seccion Servicios/Subempresas en Kumera Web

Fecha: 2026-02-25

## Contexto
Se posterga para la siguiente iteracion la incorporacion de una seccion visual en `kumeraweb.com` que muestre los servicios/subempresas del ecosistema Kumera.

## Objetivo del proximo paso
Agregar una seccion institucional en la web oficial de Kumera para presentar:

1. TuEjecutiva
2. Tractiva
3. LeadOSKu (pendiente de logo/publicacion)

## Alcance propuesto

1. Bloque "Servicios operados por Kumera" en home.
2. Tarjetas por servicio con:
   - Nombre
   - Descripcion corta
   - Estado (activo / proximo lanzamiento)
   - CTA (visitar web o "proximamente")
3. Logos:
   - TuEjecutiva: usar logo actual.
   - Tractiva: usar logo actual.
   - LeadOSKu: placeholder hasta definir branding.

## Criterios de aceptacion

1. La seccion aparece en home y es responsive.
2. No rompe performance ni layout actual.
3. Permite agregar futuros servicios sin refactor mayor.
4. Mantiene coherencia de marca corporativa Kumera.
